# 🧠 Quiz Management System

A modern desktop Quiz Management System built with C# and Windows Forms using .NET. QuizManagement is designed to provide an interactive quiz experience for students while allowing teachers to manage quizzes through a simple and organized interface.

---

## 🚀 Technologies Used

- C#
- .NET 8
- Windows Forms (WinForms)
- SQL Database
- MSTest (Unit Testing)

---

## ✨ Features

- Student and teacher login system
- Interactive quiz interface
- Automatic quiz scoring
- Display quiz results
- Teacher quiz management
- Organized project structure
- Simple and user-friendly UI

---

## 👤 What Users Can Do

### Students
- Log into the application
- Take quizzes
- Submit answers
- View quiz results and scores

### Teachers
- Log into the teacher portal
- Manage quiz questions
- Create and organize quizzes
- Review quiz functionality

> **Note:** This project is a desktop application built with Windows Forms and is intended for learning purposes. Additional features such as online authentication and cloud storage are not included.

---

## 🛠️ How I Built It

I built this project using C# and Windows Forms in Visual Studio. The application was structured into separate forms for students, teachers, quizzes, and results to keep the code organized and maintainable. A database was used to store quiz information, while helper classes handled scoring and UI functionality. I also included unit tests using MSTest to improve code reliability.
